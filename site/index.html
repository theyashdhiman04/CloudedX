Overview

CloudedX is an Infrastructure-as-Code solution that automates the deployment of a production-ready Kubernetes monitoring stack on AWS. It provisions networking, compute, and a complete Prometheus + Grafana stack for end‑to‑end observability.

Key Features

  • Automated AWS infrastructure using Terraform
  • Self-managed Kubernetes cluster (control plane + worker nodes)
  • Integrated Prometheus and Grafana monitoring stack
  • Modular network and compute Terraform modules
  • Sample application to validate the cluster

Quick Start

  1. Clone the repository from GitHub.
  2. Configure your AWS credentials and SSH key pair.
  3. Run terraform init, terraform plan, and terraform apply.
  4. Follow DEPLOYMENT.md to finish Kubernetes and monitoring setup.

Technology Stack

CloudedX is built using:

  • Terraform for Infrastructure-as-Code
  • AWS for compute and networking
  • Kubernetes for orchestration
  • Prometheus and Grafana for metrics and visualization
